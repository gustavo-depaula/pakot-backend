Rotas relacionadas a entregadores

Tipo de requisição Rota enviada Função Parâmetros enviados Retorno
POST '/DeliveryMan/getData' Retorna os dados de um entregador a partir de um email. email JSON(entregador)/string 'noDataFound'
POST '/DeliveryMan/getAssigned' Retorna os dados de pacotes que foram atribuídos a um entregador a partir de seu email. email JSON(array com pacotes)
POST '/DeliveryMan/getDone' Retorna todos os pacots já entregues de um entregador. email JSON(array com pacotes)
POST '/DeliveryMan/assignPackage' Atribui um pacote a um entregador a partir de seu email e id. email, id string com confirmação de ação.
POST '/DeliveryMan/getGainData' Retorna todos os ganhos de um entregador. email JSON(objeto com todos os dados de ganhos).
POST '/DeliveryMan/emails' Retorna uma array com todos os emails cadastrados como entregadores. JSON(array com emails).
POST '/DeliveryMan/updateWallet' Atualiza o valor da carteira de um usuário. email, value string com confirmação de ação.
POST '/DeliveryMan/getWallet' Retorna o valor da carteira de um usuário. email JSON(valor).
POST '/DeliveryMan/done' Finaliza uma entrega, e faz as transações de dinheiro. email, id string com confirmação de ação.
