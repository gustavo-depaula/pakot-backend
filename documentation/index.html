Documentação Pakot

Desenvolvimento

O lado backend do serviço pakot foi desenvolvido em PHP, com o uso da framework SlimPHP.

Com o uso de tal framwork, o funcionamento da aplicação se baseia no roteamento de requisições feitas pelo frontend.

A documentação da parte do backend é feita de seguinte forma:

Tipo de requisição Rota enviada Função Parâmetros enviados Retorno
POST '/package/create' Função de criar pacotes no banco de dados, e de linkar o cliente com o id do pacote. email, nickname, description, priority, size, weight, origin, destination JSON(pacote criado + preço)
