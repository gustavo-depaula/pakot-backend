Rotas relacionadas a pacotes

Tipo de requisição Rota enviada Função Parâmetros enviados Retorno
POST '/package/create' Cria pacotes no banco de dados, e linka o cliente com o id do pacote recém-criado. email, nickname, description, priority, size, weight, origin, destination JSON(pacote criado + preço)
GET '/package/getopen' Retorna os pacotes que não foram receberam um entregador. JSON(array com pacotes em aberto)
GET 'package/getpackage/{id}' Retorna os dados de um pacote a partir de seu id. id JSON(pacote)
POST '/package/getallpackages' Retorna os pacotes que estão relacionados a um usuário. email JSON(array com pacotes)
POST '/package/update' Altera algum estado específico de um pacote. id, status, value String com confirmação da alteração.
POST '/package/price' Retorna o preço de uma entrega a partir de suas condições. priority, distance, size, weight JSON(preço)
