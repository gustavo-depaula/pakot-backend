Rotas relacionadas a usuários

Tipo de requisição Rota enviada Função Parâmetros enviados Retorno
POST '/User/getData' Retorna informações de um usuário a partir de um email. email JSON(usuário)/string 'noDataFound'
POST '/User/update' Atualiza as informações de um usuário. email, nome, cpf, phone, payment JSON(usuário)/string 'noDataFound'
POST '/User/emails' Retorna uma array com todos os emails cadastrados como usuários. JSON(array com emails).
POST '/User/updateWallet' Atualiza o valor da carteira de um usuário. email, value string com confirmação de ação.
POST '/User/getWallet' Retorna o valor da carteira de um usuário. email JSON(valor).
